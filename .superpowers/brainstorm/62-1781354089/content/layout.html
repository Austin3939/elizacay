Product Detail Page — Layout

Which layout feels right for the shop's product pages?

ARTWORK

A — Split Panel

Large artwork fills the left half. Title, price, variants, and Add to Cart on the right. Image stays fixed while details scroll. Matches the feel of the existing Design page.

ARTWORK — FULL WIDTH

B — Stacked / Mobile-first

Image spans the full width at top, content below in a single column. Works well on mobile first. Simpler layout, less dramatic on desktop.

MAIN IMG

C — Gallery + Detail

Like A but with thumbnail strip below the main image for multi-image products. Best for Shopify products that have several photos. Slightly more complex to build.
