Product Page — Content Sections

What information appears on the product detail page?

IMAGE

A — Buy section only

Image → Type tag → Title → Price → Variant pills → Add to Cart. Clean and fast. Fits the minimal aesthetic of the site.

IMAGE

B — Buy section + Description

Everything in A, plus a description section below the Add to Cart button. Great for telling the story of the piece — medium, inspiration, size details.

IMAGE

C — Buy + Description + Accordions

Everything in B, plus collapsible rows for "Shipping & Returns", "Size Guide" (apparel), or "Print Details". More info without cluttering the page.
